## Getting Started

### Prerequisites
- Node.js (v24 or newer)
- npm or yarn

### Installation
```bash
git clone https://github.com/sbjame/myportfolio.git
cd onlineShop-dummyJson
npm install
```
### Running Locally
```bash
npm run dev
Then open http://localhost:5173 in your browser.
```
